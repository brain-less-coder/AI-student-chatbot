# AI Student Assistant Chatbot

A basic chatbot built using Flask, TensorFlow, and NLP techniques that answers student-related queries such as:

- Python roadmap  
- Machine learning roadmap  
- Internship tips  
- Project ideas  
- Resume guidance  
- Coding help  
- Motivation support  

---

## Tech Stack

- Python  
- Flask  
- TensorFlow / Keras  
- NLTK  
- HTML  
- CSS  
- JavaScript  

---

## Features

- Simple chatbot UI  
- Intent-based response system  
- Tokenization and lemmatization  
- Bag of Words implementation  
- Neural network model for intent classification

## How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/AI-Student-Assistant.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Train the model:

```bash
python new.py
```

Run the project:

```bash
python app.py
```

Open in browser:

```bash
http://127.0.0.1:5000
```
